Git Command List

Install Git

Before you start using Git, you have to make it available on your computer. You can download it from the Git website. Just go to http://git-scm.com/download/win/ and the download will start automatically.

git push

Update remote refs using local refs, while sending objects necessary to complete the given refs.

git log

This command is used to show commit logs

git init

Before begin using git, create repository structure using command $git init
