Git Command List

Install Git

Before you start using Git, you have to make it available on your computer. You can download it from the Git website. Just go to http://git-scm.com/download/win/ and the download will start automatically.

git push

Update remote refs using local refs, while sending objects necessary to complete the given refs.

git log

This command is used to show commit logs

git init

Before begin using git, create repository structure

Command:

$git init

Create File

Create a file using command echo "test" > filename or with any text editor of your choice.

git add

This command updates the index using the current content found in the working tree, to prepare the content staged for the next commit .

git push

Update remote refs along with associated objects
https://git-scm.com/docs/git-push

git pull

Fetch from and integrate with another repository or a local branch

Command:

git pull [NICKNAME] [BRANCH]

git pull

Fetch from and integrate with another repository or a local branch

Command:

git pull [options] [<repository> [<refspec>…​]]

git status

Show the working tree status

Command:

git status [<options>…​] [--] [<pathspec>…​]

git add

Add file contents to the index

Command:

git add [--verbose | -v] [--dry-run | -n] [--force | -f] [--interactive | -i] [--patch | -p] [--edit | -e] [--[no-]all | --[no-]ignore-removal | [--update | -u]] [--intent-to-add | -N] [--refresh] [--ignore-errors] [--ignore-missing] [--chmod=(+|-)x] [--] [<pathspec>…​]
https://git-scm.com/docs/git-add

branch

  • Create a new branch using command git branch branch_name
  • Switch to the new branch using command git checkout branch_name
  • Combine two above commands in one using git checkout -b branch_name
  • See all branches using command git branch
  • Delete a branch using command git branch -d branch_name

git checkout

Updates files in the working tree to match the version in the index or the specified tree.

stash

Keep your staging changes in a temporary branch using command git stash

After you finish with the working tree and want to restore the changes then use command git stash pop
